# ElectivoIA

Sistema integral para la gestión eficiente de la oferta y disponibilidad de cursos electivos en un colegio.

- 🤖 Recomendaciones personalizadas con IA
- 👨‍🎓 Inscripción autónoma de estudiantes
- 👩‍💼 Gestión de cursos, cupos y profesores
- 📊 Reportes inteligentes para administración

Tecnologías: Reflex (Python), PostgreSQL
